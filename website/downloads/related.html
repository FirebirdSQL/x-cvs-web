Migration Tools

Tools that allow developers to migrate from older versions of InterBase to newer versions, or tools that allow migration from one database to another. Also included are InterBase metadata synchronisation tools.

DATA
  • Microsoft Access and Microsoft SQL Server to InterBase Wizard V.14
    This wizard will migrate MS SQL databases and Access databases to InterBase/Firebird.
    PDF document re. MSSQL-to-GDB issues available.
    By Marcelo Lopez Ruiz
    Freeware
  • Pdox2IB Utility
    For converting a Paradox or dBase database to InterBase. Program creates a DDL script, generators, indices, allows editing before running the script to create the objects and has a datapump option. Takes care of datatype conversions, which the user can modify. The Info file is a How-to and can be invoked from the toolbar. Careful attention is needed to doing things in the right sequence. BDE and a valid database alias are needed for reading the Paradox or DBase tables. The conversion, scripting and pump processes use IB Objects.
    Also available, an article (in Winhelp format) from the IB Objects Tech Info series, about the issues involved in moving desktop database applications to Client/Server.
    By Helen Borrie
    Freeware
  • IBLoad
    Utility to load ASCII files into InterBase.

    "InterBase DataPump allows you to pump data from any BDE/ODBC sources (dBase, Paradox, MSSQL, Sybase, Oracle etc) into InterBase databases easily, quickly and painlessly. This program will help you to migrate from InterBase 5.x and older InterBase databases to the new InterBase 6.0 format. You can also use it to pump data from one InterBase database to another."
  • K2Pump
    Pumps data from any source into InterBase, if the tables don't exist k2Pump will create them for you along with the index structure. The main goal of this tool is to migrate an entire Xbase system to InterBase quickly and easily. Uses the BDE/ODBC via BDE. Source code is included (Delphi 5).

Development Components

  • FIBPlus V4.7
    FIBPlus is a library of TDataset-based components for Delphi, and also C++Builder, connecting directly to the InterBase API. FIBPlus is based on FreeIBComponents originally written by Gregory Deatz. It was first released in March, 1999 in Russian-language versions. The latest version of FIBPlus has English documentation and full InterBase V6.0 support in the Delphi versions.
    FIBPlus features fully functional support for arrays and new Dialect 3 data types, cached updates, 'correct' local data filtering, client-side support for domains and generators, control and monitoring of the data-fetching process, SQL macro support, additional client events during transaction work, pessimistic locking, capability to save connection options in a system registry, in-place row insertion, smart response to lost connections, etc.
    FIBPlus includes TDataSetContainer to centralize events handling of several TpFIBDataSets, TpFibErrorHandler to centralize error handling, TpFIBUpdateObject for creating complex queues of data updating in live queries. A 'data vocabulary' can be created to enable automatic filling of the TFields properties.
    Buy this product

    FIBPlus Enterprise License $989.00 (Unlimited Number of Developers)

    By Serg Vostrikov Devrace

  • IB Objects
    Powerful, mature series of data access components and visual controls providing direct access to InterBase API (no BDE, no ODBC) from clients built with 32-bit Delphi and C++Builder.
    'TIB_' series built on a server-centric connectivity architecture requires only 'standard' versions of Delphi and C++Builder. Freed from the memory limitations of TDataset, it gives finest level transaction and concurrency control, live queries, soft commit, specialized client/server controls for code-free state handling, incremental search, client-side handling of generators, management of Oldest Active Transaction, Query-by-Example, global data property handling by domain or datatype without the need to build a data dictionary, callback control of data fetching, 'smart buffering' for intelligent filtering, locates and in-place row insertion, full support for all data types, including InterBase array types, pessimistic locking, insertion-in-place, sychronization of datasets after updates and more.
    TDataset-compatible 'TIBO' series allows you to convert and run existing BDE applications in minutes, keeping compatibility with 3P components such as QuickReport, TeeChart, InfoPower, etc. (Needs 'Pro' versions of Delphi and C++Builder). 'Native IBO' data access is wrapped around TDataset descendants to add the server-centric capabilities to TDataset compatibility.
    Source-code is freely available - partial source with, full source on registration.
    Supports all versions of InterBase from V4.x, all 32-bit Delphi versions, all C++Builder versions from V2.
    By Jason Wharton
    Trustware
